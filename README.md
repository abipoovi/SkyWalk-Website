✈️ SkyWalk Travel Website

A modern and responsive travel website built using HTML, CSS, and JavaScript. SkyWalk helps users explore popular destinations, discover travel experiences, and enjoy an interactive browsing experience through smooth animations and engaging UI design.
🚀 Live Demo

🔗 Live Website: https://sky-walk-website.vercel.app/

📸 Features

Responsive Navigation Bar
Hero Section with Attractive Landing Page
Popular Destinations Showcase
Interactive Feature Cards
Travel Information Section
Animated Statistics Cards
Discover Section with Travel Highlights
Customer Reviews & Testimonials
Responsive Footer
Smooth Scroll Animations using AOS
Mobile-Friendly Design

🛠️ Technologies Used

HTML5
CSS3
JavaScript
AOS (Animate On Scroll)
Font Awesome
